[ ⚠️ Acesso Bloqueado ]

Sua API_KEY no arquivo .env está vazia, muito fraca, ou usando o valor padrão. Para a sua segurança, o sistema inteiro está bloqueado.

1. Abra o arquivo .env na pasta do projeto.
2. Mude para uma senha sua (mínimo 6 caracteres):
API_KEY=sua-senha-aqui
3. Reinicie o servidor no terminal.
Recarregue a página (F5) após reiniciar o servidor.
